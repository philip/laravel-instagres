# Documentation

This folder holds reference material for [philip/laravel-instagres](https://github.com/philip/laravel-instagres). Start with the [README](../README.md) for install and quick start.

- **[commands.md](commands.md)** lists every Artisan flag, copy-paste examples, sample `instagres:claim-url` output, and common workflows.
- **[configuration.md](configuration.md)** explains `config/instagres.php`, `INSTAGRES_*` variables, `.env` layouts, and pooled versus direct URLs.
- **[programmatic.md](programmatic.md)** covers the `Instagres` facade and `InstagresException` handling.
- **[upgrading.md](upgrading.md)** describes breaking changes when you move to newer package versions.
